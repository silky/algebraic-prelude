algebraic-prelude
=================

Experiments in building a minimal Haskell Prelude modeled on abstract algebra. Please don't use this.

Uses only benign language extensions:

* ``GADTs``
* ``RankNTypes``
* ``MultiParamTypeClasses``
* ``FlexibleInstances``
* ``FlexibleContexts``
